# OpoRF Aragón - versión GitHub Pages

Versión simple y estática para preparar la oposición de **Radiofísica Hospitalaria en Aragón** desde iPhone.

## Acceso

La pantalla de entrada no muestra credenciales de ejemplo. La validación se realiza localmente en el navegador mediante hashes. Al ser una app estática de GitHub Pages, no debe considerarse seguridad real frente a inspección del código; para seguridad real hace falta backend.

## Cómo publicarla en GitHub Pages

1. Sube todos estos archivos a la raíz del repositorio.
2. En GitHub: `Settings` → `Pages`.
3. En `Source`, elige `Deploy from branch`.
4. Selecciona `main` y `/root`.
5. Abre la URL que te dé GitHub Pages.

En iPhone: abre la URL con Safari → Compartir → Añadir a pantalla de inicio.

## Limitaciones

- No hay backend ni sincronización real entre dispositivos.
- Los datos se guardan en localStorage del navegador.
- Importa CSV, no Excel nativo.
- Para monetización, usuarios reales y suscripciones hace falta backend.
